🎯 **What:** The `AddServerModal` React component in `src/renderer/components/add-server-modal.jsx` was untested. This PR adds a comprehensive test suite using Vitest and `react-dom/client` `createRoot` rendering logic that follows existing testing conventions in the codebase.

📊 **Coverage:** The new test suite covers:
- Rendering the modal when `open` is true, and rendering nothing when closed.
- Handling form updates correctly, verifying that state aligns with inputs.
- Validating the form to ensure the "Add server" button is disabled if `id` or `host` is empty.
- Calling the `window.api.ssh.testConnection` IPC mock on successful and failed tests, and updating the probe status button text appropriately.
- Firing the `onAdd` callback correctly when the "Add server" button is clicked.
- Calling `onClose` appropriately when the "Cancel" button, close icon, or Escape key is triggered.

✨ **Result:** Enhanced the reliability and maintainability of the application by ensuring the critical `AddServerModal` component's behavior logic is now thoroughly verified and regressions will be caught by `npm run check`.
